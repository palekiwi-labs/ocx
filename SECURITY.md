## Reporting a Vulnerability

We take the security of OCX seriously. If you have found a vulnerability, please do not report it in the public issue tracker.

### How to Report

Please email **dev@palekiwi.com** with the subject line "Security Vulnerability in OCX".

Include as much detail as possible:
- Type of issue (e.g., privilege escalation, injection, etc.)
- Steps to reproduce
- Affected versions
- Potential impact

### Response Timeline

We are committed to addressing security issues promptly.
- You will receive an acknowledgment within **48 hours**.
- We aim to provide a preliminary assessment within **1 week**.
- We will keep you updated as we work on a fix.

### Disclosure Policy

We ask that you give us a reasonable amount of time to fix the issue before making it public. We will coordinate the public disclosure with you.

Thank you for helping keep OCX secure!
